# Contribution Guidelines

Please read this before opening a pull request.

## What belongs here

- A public project made by a Filipino builder or team.
- A working app, game, site, tool, or other useful creative project—not just prompts, a tutorial, or a template.
- AI-assisted, prompt-led development was a meaningful part of its implementation. Say how in the PR.
- A source repository, live project, or public builder profile that lets people verify the work.

## What does not belong

- Generic global projects with no Filipino builder or team.
- Empty repos, copied templates, or landing pages with no product.
- Unverified claims that a project was vibe-coded.
- AI-generated link dumps.

## Pull request checklist

- [ ] Link works.
- [ ] Description says what the project does and who it is for.
- [ ] PR explains the project's Filipino connection and AI-assisted development.
- [ ] Entry is in alphabetical order.
- [ ] No duplicate of an existing entry.

## Description style

Bad:

> Revolutionary AI-powered platform built with the latest models.

Good:

> Expense-splitting app for group trips. The builder used Cursor to turn a weekend prototype into a deployed PWA.
