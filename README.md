## Mobile apps

- **[Kabi — Gym Bro](https://apps.apple.com/us/app/kabi-gym-bro/id6763519763)** — Offline-first gym log for routines, workouts, food, and progress. It follows Bryl Lim's published Codex-and-template build playbook ([case study](https://tap-and-swipe.com/episodes/bryl-lim-tarsi)).
- **[Mayi — Travel Buddy](https://apps.apple.com/us/app/mayi-travel-buddy/id6762192203)** — Offline-first trip planner with itineraries, bill splitting, and local AI features. Bryl Lim used the same published Codex-based app workflow ([case study](https://tap-and-swipe.com/episodes/bryl-lim-tarsi)).
- **[Tarsi — Budget Tracker](https://apps.apple.com/us/app/tarsi-budget-tracker/id6760278399)** — Offline-first personal-finance tracker with on-device AI. Bryl Lim vibe-coded its first version with Codex and a React Native template over one weekend ([case study](https://tap-and-swipe.com/episodes/bryl-lim-tarsi)).

## Data & civic

- **[2025 National Demographic and Health Survey Dashboard](https://tordecilla.github.io/ph-ndhs-2025-web/)** — Interactive dashboard for Philippine health-survey indicators. Jaemark Tordecilla built it with coding agents after a week of vibe coding ([build notes](https://reutersinstitute.politics.ox.ac.uk/news/i-vibe-coded-complex-data-visualisation-and-analysis-dashboard-heres-what-i-learned)).
- **[SentroLink](https://www.linkedin.com/posts/viron-gil-estrada_nagacityhackathon-sentrolink-ai-activity-7417030199457927168-0EL6)** — Health-triage prototype for the MyNaga app that routes residents to the appropriate care. Its builder says the team reached a working prototype in 48 hours through vibe coding ([build notes](https://www.vibecoders.ph/blog/death-of-tutorial-hell)).

## Web & community

- **[Gen AI to Z](https://www.vibecoders.ph/events/gen-ai-to-z)** — Event hub, registration, check-in, and reporting system for Vibe Coders PH's 2026 UP Diliman career summit. Its organizer documents vibe-coding the site and operations tooling with a Supabase backend ([build notes](https://www.vibecoders.ph/blog/gen-ai-to-z-post-summit-letter)).
- **[Vibe Coders PH](https://www.vibecoders.ph/)** — Filipino builder community with a public project showcase. Its author documents shipping the showcase in four hours with Supabase MCP and vibe coding ([build notes](https://www.vibecoders.ph/blog/ai-engineer-survival-guide-supabase-mcp-vibe-coding)).
